Card

{% code 'django' %}{% card card_id="card-example" image=card_image02 url='#' title='Lorem ipsum' description='Lorem ipsum dolor sit amet...' ... other params %}{% endcode %}

Possible params:

Name Expected value Description
card_id String Hyphenated string to use as the HTML ID of the element
title String The title of the card
description String Long description for the card
subheading String A smaller heading to display above the title
image URL The image shown at the top of the card
no_padding Boolean True = text extends to the very edge of the card
transparent Boolean True = no background colour is used
