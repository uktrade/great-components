Responsive classes

All classes are mobile first.

Add the following suffixes to apply the styles below only to specific screen sizes:

Size Suffix
Tablet and up -m
Desktop and up -l
Extra wide screens -xl

E.g. to make an element â…” width only on tablet you would add the class w-2-3-m. Or for a grid column use c-2-3-m.

Conditional display

Use these classes to show/hide elements according to screen size. Bear in mind this will also hide the elements for screen readers.

.display-mobile
.display-tablet
.display-desktop
