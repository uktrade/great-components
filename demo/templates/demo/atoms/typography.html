Typography

Utility classes

Class name Description
.font-brand Styles text with the main brand typeface.
Note: we only use this typeface in one weight which is bold (700).
.font-body Styles text with the body typeface.
.italic Makes text italic.
.bold Makes text bold.
.uppercase Makes text uppercase.

Headings

Use these heading classes to change the size of headings while still keeping HTML semantic. To match up with the base line height, each heading takes up the height of a different multiple of the base line height according to its font size.

Class Number of lines
.h-xl 3
.h-l 3
.h-m 2
.h-s 2
