Spacing

Spacings are based on the line height at base font size. Classes that use these spacings will switch sizes according to screen width to keep spacing proportional to the text size.

To see how elements on this page line up with the base line height, use the button in the bottom right to toggle the line height guide.

For further reading see webtypography.net.

Suffix Number of lines px value (desktop) px value (mobile)
-xxs ¼ 7.5px 5.5px
-xs ½ 15px 11px
-s 1 30px 22px
-m 1½ 45px 33px
-l 2 60px 44px
-xl 3 90px 66px

Padding and margin

Rule Prefix
Margin m-
Padding p-
Position Abbreviation
All sides
Top t
Bottom b
Right r
Left f
Top and bottom (vertical) v
Left and right (horizontal) h

To construct a class, take the rule abbreviation, add the position, then the spacing size. E.g. for a small margin on the top and bottom use m-v-s. For a large padding on the left use p-f-l. For a medium padding on all sides use p-m
